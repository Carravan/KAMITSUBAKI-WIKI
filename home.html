Welcome to the KAMITSUBAKI LORE Wiki!

The aim of this project is to assemble an English source for Vsingers and related content, created by KAMITSUBAKI STUDIO.

This Wiki is a fan project of the KAMITSUBAKI STUDIO Unofficial English Community Discord server. We are not affiliated with KAMITSUBAKI STUDIO.

How to contribute

Since the origin of this Wiki is the KAMITSUBAKI STUDIO Unofficial English Community Discord Server, contributors need to be a member of the Discord Server. Members of the Discord Server are able to log in using Discord SSO.

If you would like to contribute to this Wiki, please follow the below steps.

  1. Join the KAMITSUBAKI STUDIO Unofficial English Community Discord server.
  2. Click the icon in the top-right of this page.
  3. From the Authentication Provider list, select Discord and follow the on-screen instructions.
  4. After successful authentication, you'll be redirected to the Wiki, now able to edit and create new pages.
